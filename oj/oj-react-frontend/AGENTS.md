# Agent Instructions

## Environment Setup

This project uses `nvm` to manage Node.js version (see `.nvmrc`).

**Before running any Node.js / npm / pnpm / npx commands, always run:**

```bash
source ~/.nvm/nvm.sh && nvm use
```

## Tech Stack

- React + TypeScript + Vite
- pnpm as package manager
- Tailwind CSS + shadcn/ui
- Jotai
- TanStack Router
